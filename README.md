# 🧠 AI 黑话互动词典

> 看懂 AI 圈的行话 · 一个面向非技术同学的 AI 概念速查词典

把每个 AI 概念,都想象成你新招的天才实习生「小克」的故事——用生活化的类比,把抽象的 AI 术语讲到人人能懂,并附上**产品经理视角**。

## 🔗 在线访问

- **GitHub Pages:** https://kevin17-mc.github.io/ai-glossary/

手机、电脑浏览器直接打开即可,无需安装。

## ✨ 功能特性

- 📚 **17 个高频 AI 术语**:LLM、Token、RAG、微调、Agent、MCP、幻觉、涌现等
- 🔍 **实时搜索**:输入关键词即时筛选词条
- 🏷️ **分类筛选**:按「基础概念 / 成本相关 / 进阶架构 / 前沿」等分类浏览
- 📖 **故事化讲解**:每个概念都有「小克」的类比故事,降低理解门槛
- 💼 **产品经理视角**:每个术语附带 PM 视角解读,贴合 AI 产品岗位
- 🌙 **深 / 浅色切换**:一键换肤,并自动记住你的偏好
- 📱 **响应式设计**:手机、平板、电脑自适应

## 🛠️ 技术栈

- 原生 **HTML / CSS / JavaScript**,单文件实现,零依赖
- 使用浏览器 **localStorage** 记住主题偏好
- 部署于 **GitHub Pages**

## 💡 项目初衷

很多非技术背景的同学想了解 AI,却被满屏的英文黑话挡在门外。
我希望用最通俗的"讲故事"方式,让任何人都能在几分钟内听懂这些概念,
并从产品的角度理解它们为什么重要。
